Pouches

A minecraft mod that provides 'pouches', items you can carry around with you with the inventory space of a single chest.

Recipe:

 N
LCL
 L

Where N is a gold nugget, L is a piece of leather, and C is any color dye.

Configuration:

I:pouch=<id>
  The item ID to set the pouch to

B:allowPouchInPouch=<true/false>
  If true, pouches may be placed in pouches.  Note that this allows for infinite inventory space!
